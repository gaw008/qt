## 🚀 启动指南

### 快速启动
```bash
# 启动主交易中心 (推荐)
python run_trading_hub.py

# 或者使用Streamlit直接启动
streamlit run main_trading_hub.py
```

### 模块化启动
```bash
# 高级多资产交易平台
python run_trading_hub.py --module advanced

# 专业期货交易界面
python run_trading_hub.py --module futures

# 增强实时监控系统
python run_trading_hub.py --module realtime

# AI学习进度中心
python run_trading_hub.py --module ai

# 原始增强仪表盘
python run_trading_hub.py --module original
```

### 自定义选项
```bash
# 自定义端口
python run_trading_hub.py --port 8502

# 调试模式
python run_trading_hub.py --debug

# 不自动打开浏览器
python run_trading_hub.py --no-browser
```
