## 错误处理
接口调用失败时会raise Exception，并返回错误码和错误消息

错误类别与对应的错误返回码详见下表：

|错误码    |说明|
|----|----|
|0  |请求成功|
|1  |服务端错误|
|2  |网络超时|
|3  |客户端错误|
|4  |拒绝访问|
|5  |接口限流|
|1000|  通用参数问题|
|1010|  业务参数问题|
|1100|  环球账户下单问题|
|1200|  标准账户下单问题|
|1300|  模拟账户下单问题|
|2100|  行情-股票数据问题|
|2200|  行情-期权数据问题|
|3xxx|  订阅数据问题|
|4000|  权限不足问题|
