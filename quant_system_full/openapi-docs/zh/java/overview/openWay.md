# 开通方法
您只需完成老虎证券线上开户并入金，且账户当前资产≥2000美金（或等值货币），就能够获取Open API开通权限

对于已经满足开通要求的**个人投资者**，可以访问[API权限开通页面](https://www.itiger.com/openapi/info)，完成认证后就可以开始参考本文档的[快速入门](/zh/java/quickStart/prepare.html#环境要求)部分使用Open API。

对于**机构投资者**，可以通过老虎证券机构后台完成开发者信息注册来开通使用权限，详情见[机构账户中心](https://www.tigerbrokers.com.sg/acct/login)

如果您在开通过程中遇到任何问题，请[**联系我们**](/zh/java/feedback/#官方量化qq群)
