### 概述
老虎开放平台可以为个人开发者和机构客户提供接口服务。为便于用户使用，我们开发了Open API SDK。Open API SDK中封装了老虎底层的接口，允许在用户自选的编程语言（目前支持Java/Python）编写的交易程序中方便地调用Open API接口，以使用老虎的行情、交易及推送服务。目前老虎API支持操作用户id下的全部类型的账户，包括环球账户，综合账户和模拟账户

### 面向用户
开放平台主要面向有一定的代码开发能力，同时对量化策略有一定的理解和研究的量化投资者

### 主要功能
- 直接管理交易：创建订单，修改或取消订单，以及检查订单状态
- 查看帐户信息：例如余额和当前头寸
- 查询行情变动：股票/美股期权价格等相关的行情信息
- 订阅实时变动：订单变动，持仓变动，行情变动等

### 支持多种订单类型
- Open API目前支持的订单类型有：
    - 市价单
    - 限价单
    - 止损单
    - 止损限价单
    - 跟踪止损单
    - 算法订单
        - TWAP
        - VWAP

目前具体可下单订单类型请见[FAQ - 交易 - 可下订单类型列表](/zh/java/FAQ/trade.html#支持交易的订单类型)

### 支持多种账户
- 环球账户：支持全部市场类型，账户类型包括保证金和现金，支持任意时段下单
- 综合账户：支持美股、港股和A股交易，融资融券功能支持部分股票，支持期权和期货交易，账户类型包括保证金和现金，任意时段下单
- 模拟账户：支持美股、港股和A股交易，支持期权交易

### 免费使用
用户只需完成老虎开户并满足当前资产≥2000USD就可以免费使用Open API进行交易，账户中的其它币种会自动换算，以美元计价。开通API后，如当前资产不到2000USD也不会取消API权限。通过 Open API 进行交易，没有附加收费，交易费用与通过 APP 交易的费用一致。具体交易佣金及其他收费可以参照老虎官网的[收费说明](https://www.tigerbrokers.com.sg/commissions/fees/)详细了解

*行情权限独立于APP，需要单独购买，详情请见[附录3: 收费及权限](/zh/java/appendix3/fee&permission.html)
