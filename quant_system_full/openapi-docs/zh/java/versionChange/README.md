### 1.3.0

`1.3.0` 版本发布 @2021-11-28

### New Features

- 成交记录查询接口上线
- 支持综合账户窝轮、牛熊证下单
- 新增期权盘口数据推送
- 延时行情接口上线(免费的股票行情接口)

### Bug Fixes

- 完善若干文档说明
- 修复使用代理时无法获取deviceId问题
- 合约获取接口统一为V2.0版本，返回增加shortInitialMargin，shortMaintenanceMargin，shortableCount等字段
- 期权摘要接口返回增加"最新成交时间"字段timestamp, latestTime



### 1.2.0

`1.2.0` 版本发布 @2021-11-14

### New Features

- 新增`V3.0`版本获取合约接口
- 新增支持按行情类别（`QuoteKeyType`枚举）来行情订阅的接口
- 新增`TradeOrderRequest`的下单方式，方便直接用合约对象下单
- 期权链接口支持更多筛选条件
- 新增配置对象`ClientConfig`，支持配置合区合约和下单时的默认账号，支持读取私钥文件来配置私钥

### Bug Fixes

- 优化客户端长链接管理，被其他连接踢出时主动关闭连接
- 优化订阅`callback interface`，增加回调结果的`subject`参数
